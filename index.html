PRIMER REPOSITORIO EM GITHUB

Practica para la creacion del primer repositorio usando los comandos siguientes:

  1. cd~
  2. cd desktop/
  3. mkdi *nombre del repositorio
  4. cd..
Ya dentro de la carpeta del repositorio continuamos asi :
  • git config --global user.name "nombre de ususario github"
  • git config --global user.email "email usuario github"
  • git init
  • git branch -m main
  • git add .
  • git commit -a -m "nombre del primer commit"
  • git remote add origin 'url del proyecto en github'
  • git push -u origin main
  • git >> readme.md
  • git add .
  • git commit -m 'nombre de lo realizado en el documento segun sea el ejemplo'
  • git push -u origin main

Una vez ya en el repositorio, siempre que tengamos que realizar alguna alteracion, usaremos los sigientes comandos:
  • gir add .
  • git status
  • git commit -m 'nombre del commit segun sea el caso
  • git pusch -u origin main
Para la creacion de una branch usaremos los siguientes coamdos:
  • git checkout -b 'nombre a ser asignado a la nueva branch'
  • git branch
  • git add .
  • git status
  • git commit -m 'nombre del commit segun sea el caso'
  • git pusch -u origin 'nombre del branch'

Y ya por ultimo para juntar el trabajo realizado entre las branch (principal/main o segundaria), usaremos los siguientes comandos:
  • git checkout master
  • git commit 'ejemplo: primer trabajo en conjunto'
  • git merge 'nombre del branch que debe ser adjuntada'
